# Подготовка за контролно 1

## Задача 1.

Напишете булеви изрази, които по две валидни точки от шахматна дъска проверява дали може да се достигне от първата до втората точка с един ход на топ, цар, кон или царица.

---

## Задача 2.

Напишете програма, която прочита от стандартния вход цяло неотрицателно число. Да се изведе на стандартния изход най-голямата разлика между две съседни цифри на числото.

---

## Задача 3.

Да се напише програмен фрагмент, който въвежда цяло положително
число number от клавиатурата. Кодът да връща най-голямото число, което може да се
получи чрез замяна на (всички срещания) на някоя от цифрите на number с цифра,
която не се среща в number. Ако не е възможно числото да стане по-голямо чрез
замяна на цифри, да се върне същото число.
Примери: 0 -> 9, 1182 -> 9982, 1192 -> 8892, 9871 -> 9876, 999 -> 999

## Задача 4.

Редицата наричаме зигзагообразна, ако съседните ѝ елементи са последователно по-големи и по-малки, т.е.
`a[0] < a[1] > a[2] < a[3] > ...` или `a[0] > a[1] < a[2] > a[3] < ...`
Да се намери дължината на максималната подредица в даден масив.{програмен фрагмент}

```
IN: [12, 13, 1, 5, 4, 7, 8, 10, 10, 11]
OUT: 6
```

## Задача 5.

Даден е масив с размер n. Напищете програма, която пренарежда масива, така че отрицателните и неотрицателните числа да се редуват, без да се променя относителния ред между отрицателните и неотрицателните. Резултатът трябва винаги да почва с неотрицателно число.


```
IN: [1, 2, 3, -4, -1, 4]
OUT: [1, -4, 2, -1, 3, 4]
```

```
IN: [-5, -2, 5, 2, 4, 7, 1, 8, 0, -8]
OUT: [5, -5, 2, -2, 4, -8, 7, 1, 8, 0]
```
