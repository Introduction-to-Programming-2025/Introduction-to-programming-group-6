## Функции

- Note - за да използваме функция, трябва да сме я дефинирали на по-горен ред
- Ако имаме return type != void сме задължени да върнем стойност при всеки един случай.

```c
#include <iostream>
using namespace std;

void D() {
    float d = 40.5f;
    cout << "In function D\n";
}

void C() {
    double c = 30.5;
    cout << "In function C\n";
}

void B() {
    int b = 20;
    C();
    cout << "In function B\n";
}

void A() {
    int a = 10;
    B();
    cout << "In function A\n";
}

int main() {
    A();
    D();
    return 0;
}
```

##### Всяко извикване на функция създава стакова рамка, която съдържа:

- аргументи (кога се прави копие, кога достъпваме външни стойноси)
- локални променливи
- адресът за връщане
- след изпълнение на функцията рамката се изтрива(pop) **заедно със локалните промениви**
- паметта от една стекова рамка е невидима за останалите
- Когато се хвърли грешка (<stdexcept>) тя се разпространява надолу по стека, докато не намери try, catch блок, който да я хваща

```c

void C() {
    cout << "C();\n"
    throw runtime_error("Error in C()");
}

void B() {
    try {
        C();
    }
    catch (const logic_error& e) {
        cout << "Caught logic_error in B(): " << e.what() << '\n';
    }
    cout << "End of B()\n";
}

void A() {
    try {
        B();
    }
    catch (const exception& e) {
        cout << "Caught exception in A(): " << e.what() << '\n';
    }
}

int main() {
    A();
    cout << "Program continues...\n";
    return 0;
}
```
