### Задача 4

**2048**

### Задача 5

**Световно първенство по футбол**

- _да се реализира архитектурно решение_

От Фифа решават да те назначат като главен програмист на електронната система за световното през 2026 година. Вашата задача е да напишете програма, която да има следните функционалности:

- Мачовете се представят като двумерна матрица matches[N][N], представлява головете на отбор i срещу отбор j - диагоналът не се ползва, отбор не играе сам срещу себе си.
- Въвеждане на брой на отборите
- Въвеждане на всички мачове.
- Водене на класация за точките на даден отбор - За всеки мач: 3 точки за победа, 1 точка за равенство, 0 точки за загуба.
- Извеждане на N - отбора с точки голове.
- Извеждане на N - отбора с най-много отбелязани голове.
- Извеждане на N - отбора с най-много допуснати голове.
